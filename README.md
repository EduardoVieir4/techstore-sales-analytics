# 📊 TechStore Analytics - Análise de Vendas e Dashboard Executivo

## 📊 Dashboard Executivo

![Dashboard](images/dashboard_executivo.png)


## 📖 Sobre o Projeto

Este projeto simula o cenário de uma empresa fictícia do setor de tecnologia chamada **TechStore**, especializada na venda de computadores, periféricos, componentes e acessórios.

O objetivo foi aplicar conceitos de **Análise de Dados**, desde a geração e tratamento dos dados até a construção de indicadores de negócio, análises exploratórias e desenvolvimento de um dashboard executivo para apoio à tomada de decisão.

---

## 🎯 Problemas de Negócio

Durante o projeto, foram investigadas as seguintes questões:

* Quais produtos geram mais faturamento?
* Quais produtos geram mais lucro?
* Existe sazonalidade nas vendas?
* Quem são os vendedores com melhor desempenho?
* Quais regiões apresentam maior potencial de investimento?

## 🔍 Principais Insights

* O faturamento da empresa apresenta forte concentração nos produtos da categoria Computadores.
* A análise de Pareto mostrou que poucos produtos são responsáveis pela maior parte da receita.
* Existe sazonalidade clara no último trimestre do ano, especialmente em Novembro e Dezembro.
* O Sudeste concentra a maior parcela do faturamento da empresa.
* Há diferenças significativas de desempenho entre os vendedores.
* Alguns produtos apresentam margens elevadas e potencial para campanhas focadas em rentabilidade.

## 📈 Recomendações Estratégicas

* Reduzir a dependência dos produtos líderes de faturamento.
* Expandir ações comerciais em regiões menos exploradas.
* Reforçar campanhas promocionais nos meses de maior demanda.
* Desenvolver estratégias para aumentar a venda de produtos com maior margem.
* Replicar boas práticas dos vendedores com melhor desempenho.
